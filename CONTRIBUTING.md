# Contributing to the Inventory Demand Intelligence System

Welcome to the team! Since we communicate a lot on WhatsApp, please use GitHub to track the actual code and features so we don't lose anything in the chat history.

## Team Workflow

1. **Never work directly on the `master` branch.** 
2. Before you start a new feature or fix, create a new branch:
   `git checkout -b your-name/feature-name`
   *(Example: `git checkout -b ahmed/demand-forecasting`)*
3. Make your changes and commit them:
   `git add .`
   `git commit -m "Added the new forecasting logic"`
4. Push your branch to GitHub:
   `git push origin your-name/feature-name`
5. Go to GitHub and open a **Pull Request (PR)** so the rest of the team can review it before it gets merged into `master`.

## Communication
- Drop a link to your Pull Request in the WhatsApp group when it's ready for review!
- If you find a bug, create an **Issue** on GitHub first, then mention it in WhatsApp.
